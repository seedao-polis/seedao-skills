## 安全提醒 · Security notice

本仓库中的 Skill 由社区成员贡献，**未经 SeeDAO 统一安全审计**。在使用前，请：

- 仔细阅读 `SKILL.md` 和 `README.md`，理解该 Skill 会执行哪些操作、访问哪些数据、调用哪些外部服务。
- 对涉及第三方 API、凭证或个人敏感信息的 Skill 保持警惕，避免泄露密钥、Cookie、Token 等。
- 在将 Skill 接入正式环境或工作流前，建议在隔离环境（如沙箱、测试账号）中先行验证。
- 执行 Skill 附带的脚本前，请确认脚本来源和作用，避免运行来历不明的代码。

如果你发现某个 Skill 存在安全风险、隐私泄露或恶意行为，请 **不要公开提 issue**，请私下联系维护者。

---

Skills in this repository are **community-contributed and not centrally audited by SeeDAO**. Before using one, please:

- Read the `SKILL.md` and `README.md` to understand what actions, data, and external services the skill uses.
- Exercise caution with skills that access third-party APIs, credentials, or personal/sensitive data.
- Test in an isolated environment (sandbox or test account) before connecting it to production workflows.
- Review any bundled scripts before executing them; do not run code from untrusted sources.

If you discover a security risk, data leak, or malicious behavior in a skill, **do not open a public issue** — contact a maintainer privately.
