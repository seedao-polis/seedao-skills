# Community Schema

## Domain: SeeDAO

## Conventions
- File names: lowercase, hyphens, no spaces
- Every page starts with YAML frontmatter
- Use [[wikilinks]] to link between pages
- _data/ is machine-generated; edit Markdown files instead
- log.md is append-only

## Page Types
- `community` — 社区主页
- `person` — 成员页
- `event` — Event 页
- `graph` — 关系图谱
- `state` — 状态记录
- `log` — 操作日志
